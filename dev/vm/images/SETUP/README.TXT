Install BeOS Partition
----------------------
Before running the PartitionMagic Special Edition, first close all other
programs and then insert the CD into your CD ROM drive.  Windows will
automatically start the installation process.  If the automatic install
process does not start for any reason, select the setup.exe icon with
My Computer or from the RUN Start Bar item.

Under Windows 95, the PartitionMagic Special Edition will be installed onto
your hard drive, a new Program Group will be created, and the PartitionMagic
Special Edition will be launched.

To create a new BeOS partition, PartitionMagic Special Edition resizes an
existing partition smaller and creates a new BeOS partition of size 500MB,
850MB, or 1.5GB.  By default, the boot or active partition is selected and
both the used and free space will be displayed.  Another partition or drive
may be selected by clicking the "Select Partition.." button.

After you select OK, an additional prompt is displayed to confirm the
request.  Upon selecting OK again, the resize and BeOS partition
creation process is started.  A progress dialog box is displayed that
shows the overall process and each individual steps' progress.

Upon completion, the computer will be rebooted and the BeOS installation
process continues.

Under Windows NT, the PartitionMagic Special Edition will be installed onto
a floppy diskette.  After the diskette is created, insert the diskette and
reboot your computer.  Then enter CREATEBE at the prompt.


Uninstall BeOS Partition
------------------------

Under Windows 95 to uninstall BeOS and remove it's partition, select the
Remove BeOS icon in the PartitionMagic program group.  The PartitionMagic
Special Edition deletes the BeOS partition and restores the resized partition
to its original size.

After you select OK, an additional prompt is displayed to confirm the
request.  Upon selecting OK again, the BeOS partition is deleted and the
resize process is started.  A progress dialog box is displayed that
shows the overall process and each individual steps' progress.

Upon completion, the computer will be rebooted.

Under Windows NT, to uninstall BeOS and remove it's partition, insert the
diskette and reboot your computer.  Then enter DELETEBE at the prompt.

After you select OK, an additional prompt is displayed to confirm the
request.  Upon selecting OK again, the BeOS partition is deleted and the
resize process is started.  A progress dialog box is displayed that
shows the overall process and each individual steps' progress.


Uninstall PartitionMagic Special Edition
----------------------------------------
To uninstall the PartitionMagic Special Edition, select the Uninstall
PartitionMagic icon or select Add/Remove Software in the Control Panel.
This will remove the PartitionMagic Special Edition from the hard drive.
